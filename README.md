# README

vim-unite-watson.vim

## How to install

*.vimrc*

```vim
" Using Bundle
Bundle 'alpaca-tc/vim-unite-watson.vim'
Bundle 'Shougo/unite.vim'

" Using NeoBundle
NeoBundle 'alpaca-tc/vim-unite-watson.vim'
NeoBundle 'Shougo/unite.vim'
```

*Install watson*

```sh
git clone https://github.com/nhmood/watson-ruby/
cd watson-ruby
rake install
```
